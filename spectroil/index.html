Наши разработки

ИК-анализатор трансформаторного масла

Изделие предназначено для измерения концентрации ионола и карбонильных соединений в трансформаторном масле с высокой точностью в полевых условиях.

  • Портативный и удобный в использовании
  • Высокая точность измерений
  • Быстрый анализ образцов

Измерительная ячейка однократного применения

Специальная измерительная ячейка, используемая совместно с нашим ИК-анализатором.

  • Одноразовое использование для исключения загрязнений
  • Высокое качество материалов
  • Оптимальная геометрия для точных измерений
